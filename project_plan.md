# 株式会社北ノ将 公式ウェブサイト

## 1. プロジェクト概要
北海道札幌市に拠点を置く株式会社北ノ将の公式コーポレートサイト。
AI関連事業・電子書籍出版・風水コンサルティング・占いサービスなど多岐にわたるサービスを紹介し、
問い合わせは公式LINEへ誘導するシングルページ構成のウェブサイト。

**ターゲットユーザー**: 全国のAI・風水・占いに関心のある個人・法人
**コアバリュー**: 北海道から全国へ、AIと伝統の融合で新しい価値を提供

## 2. ページ構成（シングルページ）
- `/` - ホームページ（全セクション含む）
  - ヒーロー（トップビジュアル）
  - 会社概要（#about）
  - サービス紹介（#services）
  - よくある質問（#faq）
  - コンタクト・公式LINE（#contact）
  - フッター

## 3. コア機能
- [x] ナビゲーションバー（スクロール追従・スムーズスクロール）
- [x] ヒーローセクション（背景画像・CTAボタン）
- [x] 会社概要セクション
- [x] サービス紹介セクション（AIコンテンツ・電子書籍・風水・占い）
- [x] よくある質問セクション（アコーディオン）
- [x] コンタクトセクション（公式LINEへの誘導）
- [x] フッター
- [ ] お問い合わせフォーム（LINEへの誘導のみ、フォーム不要）

## 4. データモデル設計
データベース不要（静的コンテンツサイト）

## 5. バックエンド / 外部連携
- Supabase: 不要
- Shopify: 不要
- Stripe: 不要
- LINE公式アカウント: リンク誘導のみ（QRコード・友だち追加ボタン）
- お問い合わせフォーム: get_form_url で取得したURLを使用
